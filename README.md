# UniShop
> Spring Framework 를 사용한 의류 쇼핑몰 구축 프로젝트로, Member, Product, Cart, Jumun 기능을 가지고 있는 웹사이트 입니다.

## 개발 환경 설정

모든 개발 의존성 설치 방법과 자동 테스트 슈트 실행 방법을 운영체제 별로 작성합니다.

```sh
make install
npm test
```
